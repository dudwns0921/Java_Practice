# Java_Practice

## 방송통신대학교 Java 프로그래밍 교재 예제 실습 저장소

### 컨벤션

#### package 생성

- package는 각 예제마다 하나씩 생성

#### package 네이밍

- chapter[num1]_[num2]
  
  - num1
    
    - 몇 장인지 작성
  
  - num2
    
    - 작성한 package의 순서에 따라 순차적으로 증가
    
    - 모든 예제를 필수로 작성하기보다는 필요한 예제만 작성하기 위함
